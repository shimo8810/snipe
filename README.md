
# Snipe

VSCodeのCode Snipet用のCLIツール.

## 使い方
```bash
snipe
VSCode Snipet Generator

USAGE:
    snipe [OPTIONS] <PREFIX> [FILE]

ARGS:
    <PREFIX>    prefix of snipet
    <FILE>      input file

OPTIONS:
    -d, --description <DESCRIPTION>    description of snipet [default: ]
    -h, --help                         Print help information
```
